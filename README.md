# Pimp
Lightweight authentication server based on JSON Web Token.

- Express 4
- MongoDB/Mongoose
- Passport
- TODO LOGGER
- Google ReCaptcha v3
- I18n

### Tool Prerequisites

- [NPM](https://npmjs.org) - Node.js package manager.
- [Grunt](http://gruntjs.com/) - The JavaScript Task Runner.


## Install
```
  $ git clone git@THIS-REPO-TODO.git
  $ cd JWT
  $ npm install 
  $ grunt
```
